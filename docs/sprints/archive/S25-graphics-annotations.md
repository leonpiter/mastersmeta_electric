- **Веха:** Сквозная (редактор) — оформительский слой поверх схемы
- **Статус:** ✅ Готов · Дата окончания: 2026-05-26

- **Связано:** карта функционала — [docs/UI-RIBBON.md](../UI-RIBBON.md) (вкладка «Вставка»: text/line/rect/circle)

## Цель
Инструменты рисования и текста для **оформления** чертежа (не электрика): линии, простые
фигуры (прямоугольник, окружность), стрелки, текст. Редактируемые тип линии (сплошная/штрих/
пунктир), толщина и цвет. Аннотации — отдельный слой, не участвуют в связности (CLAUDE принцип 2:
это не `Wire`).

## Scope
- [x] Ядро: `Annotation` (line/rect/ellipse/text; стрелка = line+arrowEnd) + `AnnotationStyle` {цвет, толщина, тип линии} в `Page.annotations`; команды Add/Remove/Move/Restyle (обратимы); persistence (round-trip в `.esch`). +тесты (5).
- [x] Канвас: режим рисования (down→drag→up для линии/фигур/стрелки; клик для текста → модалка), ghost-превью, слой рендера, выбор/перемещение/удаление аннотаций; **пробел вращает выбранный УГО** на 90°.
- [x] Лента: вкладка «Оформление» — инструменты (Линия / Прямоугольник / Окружность / Стрелка / Текст) + стиль (тип линии, толщина, цвет); стиль применяется к новым и к выбранной аннотации.
- [x] Ввод текста — отдельная модалка (`#text-dialog`), не браузерный prompt.
- [ ] (опц.) Размерные линии (dim), редактирование текста/размера после создания, ручки resize, привязка вкл/выкл.

## Технический дизайн
- `Annotation` — дискриминированный union по `kind`; общий `style: AnnotationStyle`.
  Стрелка = `kind:"line"` с `arrowEnd:true`. Текст использует `style.color` как заливку.
- Слой аннотаций рендерится в собственной `<g>` поверх листа, но под курсором/ghost.
- Аннотации **не** участвуют в `computeNets`/`computeJunctions` — чистая графика.

## DoD
