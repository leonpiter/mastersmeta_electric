## Цель
Десктоп-инсталлятор, который сам обновляется. Решение по оболочке: **Electron** (Node уже
есть, авто-обновление из коробки; выбор обратим — UI за Platform-портом, принцип 7).

## Примечания
- Локальная сборка инсталлятора (`pnpm --filter @see/desktop dist`) на Windows без **Режима
  разработчика** падает на распаковке `winCodeSign` (символьные ссылки macOS-библиотек) — это
  ограничение прав ОС, не конфиг. CI (`windows-latest`) собирает без проблем; локально —
  включить Режим разработчика или собирать через CI.
