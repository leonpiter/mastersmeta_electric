# S1 · Лист и штамп

- **Веха:** A — Редактор схем
- **Статус:** запланирован · **ветка** `phase-1-sheet-titleblock`
- **Зависит от:** S0

## Цель
Лист чертежа по ГОСТ: формат, рамка, зонная сетка, основная надпись (штамп). Экспорт листа в PDF.

## Scope
- [ ] `core/sheet.ts`: форматы A4/A3 (мм), `frameRect` (ГОСТ 2.301: поля 20/5/5/5), `zoneGrid` (колонки 1…N, ряды A…M), модель `TitleBlock` (ГОСТ 2.104, Форма 1) + `defaultTitleBlock`.
- [ ] Расширить `Page`: `format`, `titleBlock` (обновить `createPage`).
- [ ] `desktop` рендер: бумага, рамка (внешняя тонкая + внутренняя толстая), зонные тики/метки, основная надпись (data-bound).
- [ ] `fitView` — вписать лист в окно.
- [ ] Экспорт текущего листа в PDF (печать браузера или `svg2pdf`/`pdf-lib`).
- [ ] Snapshot-тест геометрии листа/рамки.

## Технический дизайн
- Линии листа — в мм (масштабируются с зумом → печать в реальных мм).
- Штамп — **шаблонный, data-bound**: наименование, обозначение, организация, Разраб./Пров., литера, масса, масштаб, лист/листов.
- Геометрия (frame/zones) — в `core` (headless); SVG-рисование пока в `desktop` (позже вынести в `core/render`).
