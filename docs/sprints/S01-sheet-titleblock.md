# S1 · Лист и штамп

- **Веха:** A — Редактор схем
- **Статус:** ✅ Готов · **ветка** `phase-1-sheet-titleblock` · завершён 2026-05-25
- **Зависит от:** S0

## Цель
Лист чертежа по ГОСТ: формат, рамка, зонная сетка, основная надпись (штамп). Экспорт листа в PDF.

## Scope
- [x] `core/sheet.ts`: форматы A4/A3 (мм), `frameRect` (ГОСТ 2.301: поля 20/5/5/5), `zoneGrid` (колонки 1…N, ряды A…M), модель `TitleBlock` (ГОСТ 2.104, Форма 1) + `defaultTitleBlock`.
- [x] Расширить `Page`: `format`, `titleBlock` (обновить `createPage`).
- [x] `desktop` рендер: бумага, рамка (внешняя тонкая + внутренняя толстая), зонные тики/метки, основная надпись (data-bound).
- [x] `fitView` — вписать лист в окно.
- [x] Экспорт листа в PDF (кнопка «⎙ PDF» → печать браузера → Save as PDF, размер 1:1 мм).
- [x] Тесты геометрии листа/рамки (`sheet.test.ts`: рамка A3/A4, зоны 8×6).

## Технический дизайн
- Линии листа — в мм (масштабируются с зумом → печать в реальных мм).
- Штамп — **шаблонный, data-bound**: наименование, обозначение, организация, Разраб./Пров., литера, масса, масштаб, лист/листов.
- Геометрия (frame/zones) — в `core` (headless); SVG-рисование пока в `desktop` (позже вынести в `core/render`).

## Открытые вопросы
- Точные размеры граф Формы 1 — уточнить по ГОСТ 2.104 (первый проход — приближённая разметка).
- Тулбар/настройки клетки (будущее, **не первостепенно**): кнопка вкл/выкл, выбор шага (0.1…50 мм, как в See Electrical), **яркость/прозрачность клетки** в настройках. Флаг `gridVisible` и `page.gridStep` уже data-driven — нужен UI-контрол (напр. в S2 или отдельной задаче «Настройки канваса»).
