# S21 · UI-оболочка: лента, доки, мультилист, иконки

- **Веха:** Сквозная (UI-оболочка) — наполняется параллельно функциональным спринтам
- **Статус:** 🚧 В работе · ветка `phase-2-symbols`
- **Зависит от:** S20 (гейт), S2 (символы)
- **Связано:** карта функционала — [docs/UI-RIBBON.md](../UI-RIBBON.md)

## Цель
Каркас десктоп-оболочки в стиле See Electrical: верхняя **лента** с вкладками, два боковых
дока (структура проекта / база УГО), статус-бар; единая **система иконок**; управление **листами**
(создание/удаление/переключение) в дереве проекта. Функциональные инструменты вкладок приходят из
своих спринтов (Соединения — S3, Нумерация — S4 и т.д.); S21 отвечает за оболочку и её наполнение.

## Scope (задачи)
- [x] Ребро ленты: вкладки (Файл-меню + Правка/Вид), группы с подписями, дропдауны (Файл, Сетка).
- [x] Доки: «Структура» (дерево проекта) и «Символы» (база УГО), обмен местами, ресайз границей, статус-бар.
- [x] **Управление листами**: в дереве «Структура» — добавить (+ Лист)/переключить кликом; удаление — ПКМ-меню листа или выбор+Del; `core`: `Project`/`pages`/`activePageId` + `AddPageCommand`/`RemovePageCommand` (обратимы) + `CanvasView.setPage`. +4 теста.
- [x] **Контекстное меню корня проекта** (ПКМ): Переименовать + Настройки → модальное окно настроек проекта (наименование, стандарт ЕСКД, число листов); Копировать/Удалить — заглушки до менеджера проектов (S11).
- [ ] **Система иконок**: один монохромный SVG-набор (`apps/desktop/src/icons.ts`), единый размер/обводка; перевести существующие кнопки на него; см. инвентарь ниже.
- [ ] Наполнение вкладок по мере функциональных спринтов (см. UI-RIBBON.md): Вставка, Соединения, Схема, Лист, Каталог/отчёты, Шкаф, Сервис.
- [ ] (опц.) Линейки, менеджер слоёв, тёмная тема.

## Технический дизайн
- **Иконки** — `apps/desktop/src/icons.ts`: функции, возвращающие `<svg class="ic">`; единый стиль через CSS `.ic`.
- **Мультилист** — `core`: `Project { id, name, pages: Page[], activePageId }`; команды `AddPageCommand`/`RemovePageCommand` (обратимы). `ProjectPanel` рендерит список листов с кнопками +/× и переключением; `CanvasView.setPage(page)` перерисовывает лист/символы и вписывает вид.
- Удаление последнего листа запрещено (минимум 1 лист).
