# S28 · Полноценный графический редактор УГО

- **Веха:** C — Создание контента (расширяет S9/S27)
- **Статус:** 🚧 В работе · ветка `phase-2-symbols`
- **Зависит от:** S27 (редактор pan/zoom, категории)

## Цель
Превратить редактор УГО из маленькой модалки в профессиональный графический редактор:
режим на всю область канваса (подсветка + баннер), инструменты в верхней ленте (вкладка
«Редактор УГО»), сменный шаг сетки, Esc не закрывает, ручки трансформации (ресайз/растяжение),
undo/redo, мультивыбор, копирование, поворот/зеркало, числовые свойства. Остаёмся в текущих
примитивах (линия/прямоуг./окружн./текст); эллипс/дуга — позже (правки ядра).

## Решения пользователя
- Форма: режим поверх канваса + баннер сверху (Сохранить/Отмена), подсветка фона.
- Лента: оставить вкладки + добавить «Редактор УГО» (инструменты только на ней; авто-активация).
- Примитивы: сначала довести текущие, эллипс/дуга — в будущем.

## Scope (фазами)
- [ ] **Ф1. Режим-канвас + лента + шаг сетки + Esc** — `#symbol-editor` `<dialog>`→`<div>` в `grid-area: canvas` + баннер; поля/инструменты → страница ленты `data-tab="symbol"` + скрытая вкладка; шаг сетки (1/2.5/5/10); Esc только прерывает действие.
- [ ] **Ф2. Ручки трансформации** — прямоуг. (угол=пропорц., сторона=1 ось), окружность (радиус), линия (концы); числовые свойства.
- [ ] **Ф3. Сила редактирования** — undo/redo редактора, мультивыбор, буфер (копир./дубль), поворот/зеркало, стрелки, привязки.
- [ ] **Ф4. Выводы и полировка** — переименование/числовая позиция выводов, статус-строка, курсоры/стиль ручек.
- [ ] **Ф5 (отложено)** — эллипс/дуга/полилиния (ядро + схема + рендер).

## DoD
- [ ] Редактор открывается на всю область канваса, подсвечен, инструменты в ленте; Esc не закрывает; шаг сетки меняется.
- [ ] Прямоугольник тянется за угол (пропорц.) и за сторону (1 ось); линия — за концы; окружность — радиус.
- [ ] undo/redo, мультивыбор, дубль, поворот в редакторе.
- [ ] `pnpm verify` зелёный; Playwright-приёмка по каждой фазе.
