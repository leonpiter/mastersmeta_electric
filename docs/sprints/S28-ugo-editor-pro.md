# S28 · Полноценный графический редактор УГО

- **Веха:** C — Создание контента (расширяет S9/S27)
- **Статус:** 🚧 В работе · ветка `phase-2-symbols`
- **Зависит от:** S27 (редактор pan/zoom, категории)

## Цель
Превратить редактор УГО из маленькой модалки в профессиональный графический редактор:
режим на всю область канваса (подсветка + баннер), инструменты в верхней ленте (вкладка
«Редактор УГО»), сменный шаг сетки, Esc не закрывает, ручки трансформации (ресайз/растяжение),
undo/redo, мультивыбор, копирование, поворот/зеркало, числовые свойства. Остаёмся в текущих
примитивах (линия/прямоуг./окружн./текст); эллипс/дуга — позже (правки ядра).

## Решения пользователя
- Форма: режим поверх канваса + баннер сверху (Сохранить/Отмена), подсветка фона.
- Лента: оставить вкладки + добавить «Редактор УГО» (инструменты только на ней; авто-активация).
- Примитивы: сначала довести текущие, эллипс/дуга — в будущем.

## Scope (фазами)
- [x] **Ф1. Режим-канвас + лента + шаг сетки + Esc** — `#symbol-editor` `<dialog>`→`<div>` в `grid-area: canvas` + баннер; инструменты-иконки → страница ленты `data-tab="symbol"` + скрытая вкладка; сменный шаг сетки; Esc только прерывает действие. (`6cc7db1`)
- [x] **Ф2. Ручки трансформации + два режима** — «Выбрать» (только перемещение) и «Размер» (ручки): прямоуг. угол=пропорц./сторона=1 ось, окружность=радиус, линия=концы. Шаг сетки — любой кратно 0,1 мм (пресеты 1/2.5/5/10 в datalist). _Числовые свойства — позже._
- [x] **Ф3. Линейки + направляющие (Visio)** — мм-линейки сверху/слева (следуют за pan/zoom); **сетка канваса совпадает с линейкой** (общий `pickStep`); направляющие тянутся из линеек, перемещаются, удаляются (назад на линейку / 2× / угол «мм» — все). Только в редакторе. (`c4efb20` + след.)
- [x] **Ф4. Дуга / полукруг** — примитив `arc` (cx/cy/r/a0/a1) в ядре + `arcPath`/`polarPoint`, `symbolBounds`/`validateSymbol`, JSON Schema, рендер (`symbol-render.ts` + редактор), инструмент «Дуга» (центр→радиус, ориентация по протяжке = полукруг), ручки радиуса. +тесты.
- [x] **Ф5. Сила редактирования** — undo/redo редактора (стек снимков, кнопки + Ctrl+Z/Y), числовые свойства выделенного (панель «Геометрия»), поворот/зеркало (R/M, вокруг центра габарита), сдвиг стрелками (Ф5a `35cafd0`); мультивыбор (рамка + Shift-клик + Ctrl+A), групповые операции, буфер Ctrl+C/V/D (Ф5b `2dc541d`).
- [x] **Ф6. Выводы и полировка** — переименование вывода и правка текста (строковые поля в панели «Геометрия»), числовая позиция вывода (x/y), статус-строка (курсор мм · масштаб · шаг сетки · выбор).
