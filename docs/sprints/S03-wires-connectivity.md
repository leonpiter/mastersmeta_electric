# S3 · Провода и связность

- **Веха:** A — Редактор схем
- **Статус:** запланирован
- **Зависит от:** S2

## Цель
Движок цепей (`Net`): связность вычисляется из геометрии.

## Scope
- [ ] `Wire` (≠ `Line`); рисование 1-Wire / 3-Wire; `Junction`.
- [ ] Разрез провода при вставке символа; авто-реконнект при перемещении.
- [ ] **union-find**: вычисление `Net`/потенциалов; подсветка цепи.
- [ ] Детект висящих выводов.

## Технический дизайн
- Сбор точек (pins + концы проводов + junction) → объединение по совпадению (после snap) → компоненты связности.
- `Net` не хранится — пересчитывается (CLAUDE принцип 2).

## DoD
- [ ] цепи вычисляются; перемещение тянет провода; подсветка потенциала
- [ ] тесты движка связности на нескольких топологиях
