# S5 · Master/slave и кросс-референсы

- **Веха:** A — Редактор схем
- **Статус:** 📝 Запланирован
- **Зависит от:** S3 (S2)

## Цель
Связь катушка↔контакты и перекрёстные ссылки между листами.

## Scope
- [x] `Device` **вычисляется** из инстансов по общей сигле (как `Net` из проводов; не хранится → ок с undo/redo, без миграций). Контакты КМ/K (НО 13/14, НЗ 21/22) связываются с катушкой (A1/A2) общей сиглой. `core/devices.ts`: `computeDevices`, `findDevice`.
- [x] Авто **contact cross** (зеркало контактов) у катушки с адресами листов/зон; адрес катушки у каждого контакта.
- [x] Кросс-рефы между листами; зонная адресация `lst.зона` (`zoneOf`, ГОСТ 2.104), обновляются при перерисовке (перемещение/смена листа/редактирование сиглы).
- [x] «Список контактов» (вкладка «Схема» → «Контакты»): устройства, их представления, предупреждения о несвязанных (`findUnlinked`: катушки без контактов, контакты-сироты).
- [ ] Контакты реле/контактора с **отдельной геометрией mirror-таблицы** (колонки полюсов как в See Electrical) — позже; пока список строк.

## DoD
- [x] контакт ссылается на катушку (общая сигла); у катушки авто-зеркало; кросс-рефы между листами обновляются
- [x] тесты ядра (`devices.test.ts` +6, `zoneOf` +3); визуальная приёмка через Playwright
