# S5 · Master/slave и кросс-референсы

- **Веха:** A — Редактор схем
- **Статус:** запланирован
- **Зависит от:** S3 (S2)

## Цель
Связь катушка↔контакты и перекрёстные ссылки между листами.

## Scope
- [ ] `Device` vs `SymbolInstance`; привязка контактов к катушке по общему имени + номера выводов (A1/A2, 13/14, 21/22).
- [ ] Авто **contact cross** (зеркало контактов) с адресами листов/зон.
- [ ] Кросс-рефы между листами; зонная адресация (`→3.1`).
- [ ] «Список контактов» (поиск несвязанных катушек/контактов).

## DoD
- [ ] контакт ссылается на катушку; у катушки авто-зеркало; кросс-рефы между листами обновляются
