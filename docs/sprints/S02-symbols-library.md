# S2 · Символы и расстановка

- **Веха:** A — Редактор схем
- **Статус:** запланирован
- **Зависит от:** S1

## Цель
Библиотека УГО и вставка элементов на лист.

## Scope
- [ ] Формат `*.symbol.json` + JSON Schema: `graphics[]`, `pins[]` (имя, x/y мм), `componentCode`, `kind`, `defaultProps`.
- [ ] Панель библиотеки: дерево категорий, фильтр, favorites.
- [ ] Вставка кликом / drag-drop; поворот; зеркало; привязка выводов к сетке 5 мм.
- [ ] Авто-позобозначение по ГОСТ 2.710 (QF/QS/KM/K/FU/SB/HL/M/T/XT…).
- [ ] Стартовый набор УГО по ГОСТ: QF, QS, УЗО/дифавтомат, KM, K, FU, SB, HL, M, T, XT.

## Технический дизайн
- `Symbol` рендерится из JSON в SVG (примитивы + выводы); превью в панели.
- Поворот/зеркало — трансформ инстанса; выводы пересчитываются.
