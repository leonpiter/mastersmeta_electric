# S9 · Редактор символов / блоков

- **Веха:** C — Создание контента
- **Статус:** запланирован
- **Зависит от:** S2

## Цель
Пользователь создаёт собственные УГО прямо в приложении.

## Scope
- [ ] Рисование примитивов (линия / прямоугольник / дуга / окружность); выводы на сетке 5 мм.
- [ ] Выбор поведения `Symbol.kind`: coil / component / component-aux / contact-NO/NC / terminal / black-box.
- [ ] Атрибуты символа; сохранение в `*.symbol.json` (библиотека пользователя).

## DoD
