# Справочник УГО и буквенных кодов (ГОСТ)

Справочный документ для встроенной библиотеки УГО (S31). Источники:
**УлГТУ (Свиридов, 2015)** — размеры УГО; **КГУ (Мошкин, 2013)** — Таблица А1
буквенных кодов (канон). Данные в коде: [`packages/core/src/gost-codes.ts`](../packages/core/src/gost-codes.ts).

## Методика начертания

- **Сетка выводов 5 мм** — `pin.x`/`pin.y` кратны 5. Тело символа — в мм по ГОСТ 2.747.
- **Точные размеры** (не «на глаз») — каждый УГО строится по размерам ниже и сверяется
  с образцом из документов. Несовпадение габаритов/углов/расстояний = баг.
- **Имена выводов** (IEC/ГОСТ): катушки `A1/A2`; контакт НО `13/14`, НЗ `21/22`;
  силовые `1/2`; многополюсные — `1/2, 3/4, 5/6, 7/8` (полюса в ряд, шаг 5 мм + штрих
  механической связи, ГОСТ 2.755).
- **Сигла** берётся из `SymbolDef.componentCode` → подтипы реле (KA/KT/KV…) живут в одной
  категории «Реле», но нумеруются своим префиксом.

## Таблица А1 — буквенные коды (ГОСТ 2.710)

Канон справочника (Мошкин, 2013) + 4 кода из методички УлГТУ (`SQ`, `SK`, `EL`, `KK`).
Полный список — в [`gost-codes.ts`](../packages/core/src/gost-codes.ts); ниже по группам.

- **A** — `AA` регулятор тока, `AF` регулятор частоты, `AV` регулятор напряжения,
  `AW` регулятор мощности, `AK` комплект защиты, `AKS` устройство АПВ.
- **B** — `BR` датчик частоты вращения, `BT` датчик температуры, `BW` счётчик ватт-часов,
  `BVA` счётчик ампер-часов реактивный.
- **C** — `C` конденсатор, `CB` конденсаторная силовая батарея, `CG` блок конденсаторный зарядный.
- **E** — `EL` лампа осветительная.
- **F** — `FU` предохранитель плавкий, `FV` разрядник.
- **G** — `G` генератор, `GB` батарея аккумуляторная, `GC` синхронный компенсатор, `GE` возбудитель.
- **H** — `HA` звуковая сигнализация, `HL` световая сигнализация, `HLA` табло сигнальное.
- **K** — `K` реле, `KA` токовое, `KB` блокировки, `KBS` блокировки от многократных включений,
  `KF` частоты, `KH` указательное, `KK` воспринимающая часть электротеплового реле, `KL` промежуточное,
  `KM` контактор/пускатель, `KQ`/`KQC`/`KQT`/`KQS` положения, `KS`/`KSS`/`KSV`/`KSG` контроля,
  `KST` термореле, `KT` времени, `KV` напряжения, `KW` мощности.
- **L** — `L` дроссель, `LR` реактор, `LG`/`LM` обмотки возбуждения.
- **M** — `M` электродвигатель.
- **P** — `P` прибор, `PA` амперметр, `PV` вольтметр, `PVA` варметр, `PW` ваттметр,
  `PI`/`PK` счётчики энергии, `PR` омметр, `PS` регистрирующий.
- **Q** — `Q` выключатель силовой, `QF` автоматический, `QK` короткозамыкатель, `QR` отделитель,
  `QS` разъединитель/рубильник, `QW` выключатель нагрузки, `QSG` разъединитель-заземлитель.
- **R** — `R` резистор, `RU` ОПН/варистор.
- **S** — `S` коммутация управления, `SA` переключатель/ключ, `SB` кнопка, `SF` автомат цепи управления,
  `SQ` концевой, `SK` термоконтакт.
- **T** — `T` трансформатор/автотрансформатор, `TA` тока, `TV` напряжения, `TL` промежуточный,
  `TS` электромагнитный стабилизатор.
- **U** — `UZ` инвертор/выпрямитель.
- **V** — `VD` диод/стабилитрон, `VT` транзистор, `VS` тиристор.
- **X** — `X` соединение контактное, `XN` неразборное, `XT` разборное, `XP` штырь, `XS` гнездо.
- **Y** — `YA` электромагнит.
- **Z** — `Z` фильтр индуктивный/ёмкостный.

## Каталог УГО (по группам)

Покрытие библиотеки расширяется по фазам S31. Одинаковые по ГОСТ начертания переиспользуются
(все реле = катушка-прямоугольник, все приборы = круг); отличие — в `componentCode`/категории.

- **Силовая коммутация (Q, ГОСТ 2.755)** — `Q`, `QF`, `QS`, `QW`, `QK`, `QR`, `QSG`; полюсные 1P/2P/3P/4P для QF/QS/QW.
- **Коммутация управления (S, ГОСТ 2.755)** — `SB` (НО/НЗ/НО+НЗ), `SA`, `SF`, контакты НО/НЗ/перекл.,
  с замедлением (4 типа), `SQ` концевой, `SK` термоконтакт.
- **Реле/контакторы (K, ГОСТ 2.756/2.767)** — катушка 12×6 + НО/НЗ; подкоды KA…KW, KK.
- **Приборы (P, ГОСТ 2.729)** — круг Ø10 / квадрат 10×10.
- **Пассивные (ГОСТ 2.728)** — `R` 10×4, `C`, `CB`, `L`, `LR` Ø12, `RU`.
- **Полупроводники (V, ГОСТ 2.730)** — `VD`, `VT`, `VS`, `UZ`.
- **Источники/машины (G/M, ГОСТ 2.722)** — `G`, `M`, `GB`, `GC`.
- **Сигнальные (H, ГОСТ 2.732)** — `HL`, `HA`, `HLA`, `EL`.
- **Трансформаторы (T, ГОСТ 2.723)** — `T`, `TA`, `TV`, автотрансформатор.
- **Защита (F, ГОСТ 2.727)** — `FU`, `FV`, `RU`.
- **Соединения (X, ГОСТ 2.721/2.755)** — `XT`, `XS`, `XP`, `X`/`XN`; перемычка; заземление.
- **Прочее** — `YA`, `Z`.
