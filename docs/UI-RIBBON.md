# Лента (ribbon) — карта функционала по вкладкам

Информационная архитектура верхнего меню редактора. Эталоны — See Electrical и AutoCAD
Electrical; цель — полный набор инструментов для **принципиальных** (многолинейных) и
**однолинейных** схем щитов по ГОСТ ЕСКД.

**Принципы:** «Файл» — классическое выпадающее меню; остальные вкладки — страницы ленты с
группами (у каждой группы подпись снизу). В ленте только осмысленные, рабочие действия;
по мере спринтов вкладки наполняются. Любое действие = вызов к `core` (CLAUDE принцип 1).

**Статусы:** ✅ реализовано · 🔜 ближайшие спринты (указан S#) · 🕒 позже (после MVP).

Порядок вкладок: **Файл · Правка · Вид · Лист · Вставка · Соединения · Схема · Однолинейная · Каталог и отчёты · Шкаф · Сервис**.

## Что реализовано сейчас (карта на код)

- **Файл**: Экспорт PDF (`CanvasView.exportPdf`).
- **Правка**: Отменить/Вернуть (`CommandStack`), Поворот/Зеркало/Удалить (`Rotate/Mirror/RemoveInstanceCommand`).
- **Вид**: Вписать лист (`resetView`), Сетка показ+шаг (`setGridVisible/setGridStep`), обмен панелей.
- **Вставка**: символы из библиотеки (`LibraryPanel` + `AddSymbolInstanceCommand`).

Ближайшее наполнение — по мере спринтов: **Соединения** (S3), **Нумерация/Потенциалы** (S4),
**Схема/Master-slave** (S5), **Каталог** (S6), **Отчёты** (S7), **Клеммы/Кабели** (S8).
